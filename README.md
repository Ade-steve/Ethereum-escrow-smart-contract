💼 Ethereum Escrow Smart Contract

A decentralized escrow system built using Solidity, enabling secure and trustless transactions between a client and a freelancer.

---

🚀 Overview

This project demonstrates how blockchain technology can eliminate the need for intermediaries in freelance payments.

The smart contract ensures that:

Funds are securely held in escrow
Payment is only released upon approval
Refunds are possible if conditions are not met

---

⚙️ Features

🔒 Secure ETH deposit system
👤 Role-based access (Client & Freelancer)
💸 Conditional payment release
↩️ Refund functionality
✅ Safe ETH transfer using `.call{value: amount}("")`

---

🧠 Technologies Used

Solidity (^0.8.x)
Remix IDE
Ethereum Virtual Machine (JavaScript VM)
